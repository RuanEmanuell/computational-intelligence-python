## O que é Aprendizado Hebbiano?
O Aprendizado Hebbiano é uma regra de aprendizado neural baseada na ideia de que neurônios que disparam juntos fortalecem sua conexão.

Na prática, os pesos são atualizados com base na correlação entre entrada e saída:
- Se dois neurônios são ativados juntos, a conexão se fortalece
- Caso contrário, a conexão enfraquece ou permanece igual

## Implementação
Esta pasta contém uma implementação simples em Python da Regra de Hebb, demonstrando como os pesos são ajustados com base nos padrões de entrada.

A implementação inclui:
- Um algoritmo base
- Testes com portas lógicas (AND, OR, NAND, NOR)
